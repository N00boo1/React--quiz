# React-Quiz

A Simple React Quiz App which contains the 15 MCQ’s which have 4 options in each.

## 📝 Usage

### Clone this Repo
```
git clone https://github.com/N00boo1/React--quiz.git
```
### Installation

```sh
$ npm install
```

### ✔️ Run

Then go to http://localhost:3000

```sh
$ npm start
```
